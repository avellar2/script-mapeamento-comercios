# Estratégia: CNPJ nas Mensagens e Emails

## Por que CNPJ aumenta confiança

**Sinais de confiança:**
- CNPJ = empresa formal = responsabilidade
- Pode ser consultado na Receita Federal
- Transmite profissionalismo
- Diferencia de golpista (só tem WhatsApp)

## Cuidado: Não exagerar

❌ **Ruim** (parece declaração de imposto):
```
CNPJ: 65.999.597/0001-75
Inscrição Estadual: RJ-123.456.78
CCM: 123.456.78-9
Alvará: 123456
Licença: MUNICÍPIO DO RIO DE JANEIRO
```

✅ **Bom** (limpo e suficiente):
```
CNPJ: 65.999.597/0001-75
(21) 9XXXX-XXXX
vandersonavellar.com
```

## Regras de ouro

1. **CNPJ em toda comunicação**
   - Email: assinatura
   - WhatsApp: final da mensagem
   - Proposta: cabeçalho
   - Contrato: identificação

2. **Completa com outros dados:**
   - CNPJ + telefone + site + localização
   - = Credibilidade blindada

3. **Não exagerar nos dados**
   - Só CNPJ é suficiente
   - Não precisa mostrar todos os registros

4. **CNPJ no portfolio**
   - Crie seção "Dados Cadastrais"
   - Isso mata qualquer dúvida sobre legitimidade

## Resumo

**CNPJ + telefone + site + presença local = credibilidade total**

Adicione CNPJ em:
- ✅ Assinaturas de email
- ✅ Final de mensagens WhatsApp
- ✅ Propostas comerciais
- ✅ Contratos
- ✅ Portfolio (seção Dados Cadastrais)
